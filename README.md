Project developed as part of my studies in **Visual Programming** at **IFNMG - Campus Montes Claros**.

<div align="center">

## About the project

</div>

This project is a system with a graphical interface built in C# with Windows Forms. The main goal is to make it easier to register developers and perform other related management operations.

It was a really fun project to build because I got to combine several concepts I learned in class, like creating forms, handling data, and navigating between screens.

---
<div align="center">

## Features

</div>

- Developer registration
- List of registered developers
- Edit and delete records
- User-friendly graphical interface with Windows Forms

---
<div align="center">

## Technologies used

</div>

- **C#** — main programming language  
- **.NET Framework / .NET** — development platform  
- **Windows Forms (WinForms)** — for building the graphical interface
